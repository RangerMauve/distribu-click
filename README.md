# distribu-click

A distributed "click the button" game

Live demo [here](https://distribuclick.herokuapp.com).

## Running

`npm install`

`npm run build`

`npm run start`

## Developing

Uses [Browserify](http://browserify.org).

`npm install`

`npm watch`

And run the server
